全网唯一的答案在sulbime论坛的老帖子
copy过来
1. 修改Package Control.sublime-settings
2.全文修改为
{
	"bootstrapped": true,
	"downloader_precedence":
	{
		"linux":
		[
			"urllib",
			"curl",
			"wget"
		],
		"osx":
		[
			"curl",
			"urllib"
		],
		"windows":
		[
			"wininet"
		]
	}
}
3.重启sublime2，ok了，可以下载安装插件了
